# endeavor-ne-executive-intelligence

An AI-powered executive intelligence system for Endeavor Northeast that automates research, synthesis, and executive reporting across the regional startup ecosystem.



\## Purpose



This project generates executive intelligence reports using information from:



\- Monday.com

\- Slack

\- Gmail

\- Google Calendar

\- Notion

\- Public web research



The system produces two recurring deliverables:



\- Daily Executive Slack Brief

\- Weekly Executive Intelligence Brief
